## Dev-Note
 이 웹사이트는 많은 개발자들이 개발한 다양한 프로젝트를 소개하는 공간입니다. 
 기술적 역량과 창의성을 발휘한 결과물을 웹사이트에서 공유합니다.

## 내용
- 이 사이트는 내가 사용한 기술 스택을 등록하고, 프로젝트 기간을 설정하며, 각 프로젝트에 대한 상세한 설명을 공유하는 플랫폼입니다.
